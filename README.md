# 技能→岗位推荐器 🚀
输入你的技能，基于 JD 关键词匹配，按相关性排序推荐岗位。

## 技术栈
- 后端：Flask (Python)
- 前端：HTML5 / CSS3 / JavaScript (Fetch API)
- 部署：本地运行 / 可部署到 Render / Vercel（前端）+ Railway（后端）

## 本地运行
1. 克隆项目
2. 创建虚拟环境 `python3 -m venv venv`，激活并安装 `flask`
3. 运行 `python backend/app.py`
4. 用 Live Server 打开 frontend/index.html 或直接浏览器打开
